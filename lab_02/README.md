# Laboratory Work №2  
## Data preprocessing and VHI analysis

### Description

The goal of this laboratory work is to learn how to collect, preprocess and analyze environmental data using Python.

The work consists of two parts:

1. Working with **VHI data from NOAA**
2. Working with a **power consumption dataset**

### Technologies

Python  
Pandas  
NumPy  
Matplotlib  
Jupyter Notebook

### Tasks

- Download VHI data for Ukrainian regions
- Parse and clean the dataset
- Combine data into a single DataFrame
- Create analytical queries
- Visualize data
