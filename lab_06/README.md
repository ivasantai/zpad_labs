# Laboratory Work 6 — C++ OpenCV

## Description
Program captures video from camera, displays it in a window, reacts to keyboard input and applies different image processing modes.

## Modes
- 0 — Normal
- 1 — Gray
- 2 — Gaussian Blur
- 3 — Canny
- 4 — Sobel
- 5 — Invert
- q / ESC — Exit

## Build and run

```bash
./preinstall.sh
./build.sh
./run.sh
